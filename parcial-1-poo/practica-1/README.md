Práctica 1: Clases y Encapsulamiento en PHP

Objetivo
Implementar una clase en PHP con atributos privados, constructor y métodos de acceso, siguiendo las reglas de encapsulamiento de la POO.

Descripción de la clase
Se creó la clase Usuario con:

Atributos: nombre y correo definidos como privados.
Constructor: Para la asignación inicial de datos.
Métodos: Getters y Setters para el manejo seguro de la información.

Instrucciones de ejecución
